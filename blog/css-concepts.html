Margins, Borders, and Padding...Oh, My!

January 17, 2016

Margins, borders, and padding are all styling methods in the css language. Using these methods can help set spacing inside, between, and around elements on a webpage. while conceptually relatively simple, these methods can easily weave a tangled web of confusion if you don't keep the straight.

Margins serve pretty much the function you would expect. The value of the a margin detemines the relative space between the borders of an element and other elements on a page. You can customize left, right, top, and bottom margins separately or all together at the same time.

Borders are also fairly self explanatory. They function as a physical border surrounding or encasing an element. Borders can be customized in width, color, and styled as solid, dotted, or dashed lines.

Padding is only slightly less intuitive in nature. Inside an elements borders, you may have some content like text, an image, or a piece of IKEA furniture. The padding value gives a relative amount of space between the element's borders and the content nested inside. Like margins, padding can also be tailored on the top, bottom, left, or right sides individually or all together uniformly.
