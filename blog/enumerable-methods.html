Enumerable #map

January 31, 2016

#map is a very commonly used and very powerful enumerable in Ruby. Basically, #map iterates through a data structure by passing it through a block of code that you define and then spits all the results back out into a new object. Sounds a bit confusing, but think about it like this: You have a list of items that you want to do something to. You write a block of code to perform that specific function. #map runs the list of items, one at a time, through that block and places the results into a new list lets say you have an array of city names:

cities = ["new york", "chicago", "richmond", "atlanta"]

and you want to capitalize them. You could use #map like this:

cities.map {|x| x.capitalize}

This will return a full new array of properly capitalized cities. All from one line of code!
