Adorable Arrays and Hefty Hashes

January 23, 2016

What the Hay is an Array, Anyway?

So, the Ruby language is object oriented. You can think of objects in the programming world like you would think of objects in the real world. They can take almost any form. Many time, in the real world, we use containers in which to store, sort, and organize our objects. Any time we need to access our objects, we can just pop open the storage unit, see what’s in there, and pull out what we need. One Ruby equivalent of these real world containers is called an Array. Arrays carry much of the work load for holding……well, just about anything. They can hold strings of text or characters, numbers, values, and even other containers. You name it, arrays can hold it.

A basic array might look something like this:

["this ", "is", "a", "pretty","little","array"]

The array container is outlined with square brackets. Inside the square brackets are all the tid bits of data separated by commas.

The divine creation of an array might look something like this:

my_pretty_array = ["this ", "is", "a", "pretty","little","array"]

Or maybe like this:

my_fresh_array = Array.new(["fresh","fresh","fresh!"])

How the Hay Do I Use an Array?

Each objects in an array is automatically assigned an index upon it’s addition to the array. Index numbers start at 0 and go as high as the number of objects inside or as high as the number of slots assigned to the array. So the first object in an array is at index 0. If you want to access something at a particular index in the array, you can call up this index number like this:

my_fresh_array[1]

Once you have an array filled with all your precious objects, there are a number of built in Ruby methods you can use to alphabetize, reverse, invert , convert, add, subtract, flatten,and otherwise manipulate the data in your array. The possibilities are endless!

How Bout Them Hashes?

Hashes are kind of a kissing cousin or psycho sister to arrays…or something. My point is that they are very closely related. Just like arrays, hashes are used in Ruby to store objects. Unlike arrays, hashes use key=>value pairs instead of index numbers. Keys are arbitrarily named and then assigned a specific value. Check it out:

hank_the_hash = {"relationship" => "kissing cousin"}

How Do I Use 'Em?

Just like you can use Array.new to create a new array, you can use Hash.new to create a new hash. You can also use the ‘hash literal’ method method:

literal_hash = {"key" => "value"}
				
And just like with arrays Ruby, has all kinds of built in methods that you can employ to iterate over, manipulate, and sort the objects in your hash. Very powerful, cool, hip stuff!
