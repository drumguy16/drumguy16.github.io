Sublime Text Short Cuts

Jump To Symbol

This shortcut produces a prompt that you can use to search for and jump to specific parts of your code.

command + R

Mark Word or Words

Holding this command allows you to highlight multiple selections of code and edit them all at once.

command + D

Split Screen

This is useful when working with two files that correspond with each other. Replacing 2 with another number will allow you to split into even more screens.

command + alt + 2

Go To Line

This command allows you to jump to a specific line number in your code. Maybe your debugging or developer tools specifies a line where there might be an error.

ctrl + G

Auto Indentation

This allows you to select multiple lines of code and indent them al to the same column together.

Expand Selection To Tag

selects all content between current tags (HTML/CSS). Pressing twice will grab the parent tags as well.

command + shift + A

Switch File

This allows you to switch forward and back between file tabs without using the mouse. If you know the order of tabs, you can replace the arrow keys with the numeric tab order.

alt + command + left, right, or number

Joining and Swapping Lines

This allows you to join the following line with the current without having to deal with white space or swap lines order.

command + J or Command + ctrl + up, down

Multiple Cursors

This function allows you to create multiple cursors on the line(s) above or below current position and carry out commands among all cursors simultaneously.

ctrl + shift + up, down

Add All Occurrences of the Selected Text to the Selection

Modify instances of a variable name or class globally

ctrl + command + G

Jump to Matching Brackets

Allows you to jump to the closing bracket of whatever function/loop/conditional you are in, eliminating the need to search with your naked eye.

ctrl + M

Open User Settings

This takes you to the user settings where you can override default parameters.

command + ,
